# Shelter Prep Strategic Summary

## Core Vision

Shelter Prep is a property and renovation intelligence platform connecting agents, contractors, homeowners, and investors.

It is not only estimating software, contractor leads, or a CRM. It becomes:

- property intelligence
- renovation planning
- contractor vetting
- AI-assisted project coordination
- permit and map integration
- construction analytics

## Product Direction

The core workflow is:

```text
Address -> Property Intelligence -> Verified Property Profile -> Renovation Planning
```

The first important product moment is simple: a user enters an address, sees useful property and renovation context, confirms what is true, then gets a clearer project plan.

## Priority Modules

1. Property Intake Engine
2. Verified Property Profile
3. Permit and Map Layer
4. AI Scope Builder
5. Contractor Profile and QR Intake System
6. Contractor Vetting and Trust System
7. Admin Dashboard and Analytics

## Pilot Philosophy

Reduce friction. Do not overwhelm users, require giant forms, or force complete setup before value appears.

The product should progressively gather information and create value immediately.

## Two-Week Pilot Goal

The pilot version should make three groups feel the product is useful quickly:

- Agents: enter an address and get a practical property prep view.
- Contractors: receive cleaner, more complete leads with photos, address, and scope.
- Admin: see project demand, missing info, cost signals, and estimate workflow status.

## Local Pilot

Start with Portland metro, Lake Oswego, Beaverton, and Tigard.

Pilot target:

- 10 agents
- 20 contractors
- 50 real projects

The goal is workflow learning before scale.
